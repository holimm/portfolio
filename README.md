## âš™ï¸ Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/holimm/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**

   ```bash
   yarn install
   ```

3. **Start development server**

   ```bash
   yarn dev
   ```

4. **Build for production**

   ```bash
   yarn build
   ```

5. **Run production build**
   ```bash
   yarn start
   ```
